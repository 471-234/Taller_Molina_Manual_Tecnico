# Taller Molina - Manual Técnico

## Descripción
Sistema de gestión de taller mecánico para clientes, vehículos, servicios y facturación.  
Incluye manual técnico, scripts de base de datos y documentación completa del sistema.

## Tecnologías
- C# (.NET 7 / WinForms o MVC)
- SQL Server / MySQL
- GitHub para control de versiones

## Estructura del repositorio

src/ -> Código fuente del proyecto
docs/ -> Documentación técnica y diagramas
README.md -> Portada del proyecto y descripción general
INSTALL.md -> Cómo instalar y ejecutar el sistema
DATABASE.md -> Detalle de la base de datos (tablas, triggers, SPs)
COMMENTS_GUIDE.md -> Guía de comentarios en C# y SQL
